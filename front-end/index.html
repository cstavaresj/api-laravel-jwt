Testes de Segurança

20 testes automatizados cobrindo XSS, brute force, token expirado e mais.

Rate Limiting

Endpoints da API

Rotas disponíveis em http://localhost:8000/api

Método Rota Descrição Auth
POST /api/auth/login Autenticação do usuário
POST /api/auth/register Cadastro de novo usuário
GET /api/auth/me Dados do usuário autenticado
POST /api/auth/logout Invalidar token (logout)
POST /api/auth/refresh Renovar token JWT
PATCH /api/user/profile Atualizar perfil do usuário
